Questões

Responda as perguntas abaixo:

A- Na tag < imput > com atributo type="text", para que serve o atributo "placeholder"?
O placeholder é um atributo especifico para informar o que deve ser colocado dentro de um campo de entrada/área de texto.

B- Na tag < imput > com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?
Define ou retorna o estado marcado de um botão de opção

C- Na tag < imput > para que serve o atributo "value"?
O atributo value especifica o valor de um elemento < imput >.

D- Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?
O atributo que deve ser utilizado para agrupar vários botões de rádio em um formulário é abrir e fechar o forms dentro da tag fieldset, e para que um por vez seja selecionado é necessário que o "name" seja igual em todos os "input", de forma que caracterizem eles como pertencentes ao mesmo grupo

E- Para que serve a tag < label >? Para que serve o atributo "for" nessa tag?
A tag "label" possibilita a escrita em frente ao quandrante de tipo "checkbox" e/ou "radio"; e o atributo "for" serve para conectar o input com o label, ou, o caracter com o quadrante, por isso o ID(do input) e for(do label) devem ser iguais

F- Na tag < option > para que serve o atributo "selected"?
O atributo "selected", na tag "option", define a opção selecionada como já definida primeiro antes mesmo da escolha do usuário

G- Nas tags de formulário para que serve o atributo "required"?
O atributo "required", na tag "form", serve para tornar obrigatório a resposta do usuário para a questão

H- Na tag < form > para que serve o atributo "action"?
O atributo "action", na tag form, é O programa que recebe o código

I- Na tag < form > para que serve o atributo "method"?
O atributo "method", na tag "form", serve para definir aparecer/esconder a resposta do usuário e retirar ou não o limite de caracter da URL

J- Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag < form > ?
O atributo da tag "method", na tag "form", para que a senha não apareça na URL deve ser "POST"

K- Qual valor deve ser informado no atributo "method" na tag < form > em um formulário que será usado para fazer upload de arquivos?
O atributo post caso url esteja muito grande ou queira manter privacidade e get para conferir os caminhos na url
